##### 提交一个新项目
```commandline
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/BillNwc/Gitcommand.git
git push -u origin master
```

##### 上传已存在项目
```commandline
git remote add origin https://github.com/BillNwc/Gitcommand.git
git push -u origin master
```
